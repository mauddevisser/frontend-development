# Frontend Development

Deze repository bevat frontend development opdrachten en projecten. Het doel van deze repo is het ontwikkelen van vaardigheden in moderne webtechnologieën zoals HTML, CSS en JavaScript.

---

## Inhoud

In deze repository vind je een fanpage met elementen, zoals:

- Basis HTML pagina's  
- Styling met CSS  
- Responsive design
  
---

## Tech Stack

- **HTML5**  
- **CSS3**
  
---

## Installatie

Volg deze stappen om de frontend lokaal te draaien:

# Clone de repository
git clone https://github.com/mauddevisser/frontend-development.git

# Ga naar de projectmap
cd frontend-development

# Installeer dependencies
npm install

# Start de server
npm run dev
